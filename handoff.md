3. **Server Reuse Decision**: Server reuse is determined by comparing the current server signature with cached signatures in `_SERVER_SIGNATURES`. The signature is generated by `_server_signature()` which includes:
   - inline_config
   - server_log_path

   If the signature matches a previously used server, it's reused; otherwise, a new server is started.

4. **/message Handler Location**: The `/message` handler is implemented in the `execute_query()` method of the OpenCodeExecutor class, specifically when the HTTP POST request is made to `/session/{session_id}/message`.

5. **Provider Dispatch Start**: The first point where provider dispatch should begin is in the `_dispatch_to_provider()` method, which is called after the `/message` handler receives the request and before the HTTP request is sent to the Ollama provider.
